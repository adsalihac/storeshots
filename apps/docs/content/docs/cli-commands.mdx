---
title: CLI Commands
description: Commands available in the Storeshots CLI.
---

## Generate

```bash
npx storeshots
npx storeshots generate
```

Generate screenshots for discovered Expo Router routes.

```bash
npx storeshots generate --output screenshots --platform all
```

## Routes

```bash
npx storeshots routes
```

Print every discovered route and its source file.

## Devices

```bash
npx storeshots devices
```

List supported App Store and Play Store device presets.

## Doctor

```bash
npx storeshots doctor
```

Check project structure, Expo Router directories, simulator tooling, Android tooling, and image processing availability.

## Useful options

```bash
npx storeshots generate --platform apple
npx storeshots generate --devices iphone-6-9,ipad-13
npx storeshots generate --routes /,/profile,/subscription
npx storeshots generate --dry-run
```
