---
title: Installation
description: Add Storeshots to an Expo Router app.
---

Run Storeshots directly with `npx`:

```bash
npx storeshots
```

Or install it as a development dependency:

```bash
pnpm add -D storeshots
```

Then add a package script:

```json
{
  "scripts": {
    "screenshots": "storeshots generate"
  }
}
```

## Requirements

- Node.js 20.11 or newer.
- An Expo project using Expo Router.
- An `app/` or `src/app/` directory.
- Apple simulator tooling for iPhone and iPad screenshots.
- Android Debug Bridge for Android screenshots.

Run the doctor command to inspect the local environment:

```bash
npx storeshots doctor
```
